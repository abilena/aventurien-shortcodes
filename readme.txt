[aventurien-cover]

== Description ==

You can use this plugin to display style blocks.

For example:

`
[aventurien-parchment]Place your text here[/aventurien-parchment]
`

== Installation ==

1. Upload <strong>aventurien-shortcodes</strong> folder to the <strong>/wp-content/plugins/</strong> directory.
2. Activate the plugin through the <strong>Plugins</strong> menu in WordPress.
3. That's all.
